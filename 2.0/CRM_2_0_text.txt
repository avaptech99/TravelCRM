1.  📊  Lead  Filters  (Left  Menu)  
Add  quick  filters:  
●  All  Leads  ●  My  Leads  ●  Unassigned  Leads  
 
2.  👥  User  Roles  &  Access  Control  
2.1  Agent/ Package  /  LCC (User)  
●  Access  Rules:  ○  User  can  see  all  leads  ○  Queries  can  be  assigned  by  them  to  any  user  ○  Other  users  can  view  their  queries  under  this  group  
2.1  Visa  Section(User)  
●  Access  Rules:  ○  User  can  see  only  their  own  queries  ○  Queries  can  be  assigned  by  any  user  to  them  ○  Other  users  cannot  view  their  queries  of  any  group  which  one  created  by  
them
 
2.2  Ticketing  Section(User)  
●  Access  Rules:  ○  User  can  see  only  their  own  queries  ○  Queries  can  be  assigned  by  any  user  to  them  ○  Other  users  cannot  view  their  queries  of  any  group  which  one  created  by  
them
 
2.3  Operation  Section(User)  
●  Access  Rules:  ○  User  can  see  only  booked  queries  which  one  booked  by  any  user  from  any  
group
 ○  Can  access  Actual  Cost  Section  to  edit  rest  is  view  only  
2.4  Account  Section(User)  
●  Access  Rules:  ○  User  can  see  only  booked  queries  which  one  done  by  any  user  of  any  group  ○  And  need  one  Verify  button  to  verify  the  query  
2.5  Admin  Section(User)  
●  Access  Rules:  ○  User  can  see  All  queries   ○  And  need  one  Verify  button  to  verify  the  booked  query  ○  All  admin  managed  dropdown  combo  
  
3.  🧾  In  Finalize  Booking  –  (Can  managed  by  all  group  users)  
●  In  Finalize  Booking  →  Under  Final  Quotation  No.  ○  Add  a  Dropdown  field  for  selecting  company:  ■  Skylight  ■  Travowords  ■  Travel  Window  Dubai  ■  Travel  Window  Canada  ●  Purpose:  ○  Identify  which  company  completed  the  booking  ●  Note:  ○  Dropdown  values  should  be  Admin-managed  (dynamic)
